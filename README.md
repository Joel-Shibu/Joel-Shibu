# Joel Shibu — AI Systems Engineer Portfolio

A cinematic, story-mode game-inspired portfolio showcasing AI/ML projects with GSAP ScrollTrigger animations, Lenis smooth scrolling, and picture-heavy parallax effects.

**🔗 Live Site:** https://portfolio-site-self-eta.vercel.app/

## Projects

- **NEUROSIGHT** — Browser-based neurological screening using TensorFlow.js
- **RESP-AI** — Real-time respiratory monitoring with CNN architecture  
- **AIRGUARDIAN** — Autonomous drone systems with SLAM navigation

## Tech Stack

- **Framework:** Next.js 16 + TypeScript
- **Animations:** GSAP ScrollTrigger, Framer Motion
- **Smooth Scroll:** Lenis
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## Setup

```bash
npm install
npm run dev      # Development
npm run build    # Production build
npm start        # Production server
```

## Deployment

Deployed automatically on Vercel from the main branch.
